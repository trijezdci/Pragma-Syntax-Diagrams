## Pragma Syntax Diagrams

This repository holds the image files with syntax diagrams for the
[Portable Pragma Specification for Wirthian Languages](https://github.com/trijezdci/M2-Pragmas/wiki). The image files are generated by a tcl script that is part of the specification repository (linked above). All image files are in PNG format.

This repository does not contain any code, only documentation assets!
